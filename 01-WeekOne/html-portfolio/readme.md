# Emre Koyuncu CV

This repository contains the HTML and CSS code for Emre Koyuncu's personal CV webpage.

## Overview

This project is a static webpage that showcases Emre Koyuncu's professional experience, education, skills, projects, and contact information. The page is built with HTML5 and styled with CSS, ensuring a clean and responsive design.

## Features

- **Personal Information**: Includes a personal photo, name, and contact details.
- **Professional Summary**: Brief description of Emre's professional background.
- **Education**: Lists educational background in chronological order.
- **Work Experience**: Detailed job history with descriptions of roles and responsibilities.
- **Skills**: A list of technical skills.
- **Projects**: Links to notable projects with GitHub repositories and live demos.
- **Location**: Embedded Google Maps iframe showing a location.

## Technologies Used

- HTML5
- CSS3
